🌍 À propos du projet

World2025 est une application web développée en PHP et MySQL dans le cadre de la 1ère année du BTS SIO. Ce projet vise à créer une plateforme interactive pour gérer et consulter des informations géographiques, économiques et démographiques sur les pays du monde.

L'application permet à l'utilisateur de :

  • Consulter une base de données complète des pays avec leurs caractéristiques
  • Accéder à des statistiques détaillées (population, PIB, superficie, etc.)
  • Filtrer et rechercher les pays par critères
  • Visualiser des informations comparatives entre différentes régions
Application Web Base de données Données géographiques

✨ Fonctionnalités principales

1. Gestion de base de données

Implémentation d'une base de données MySQL complète contenant les informations de tous les pays du monde : population, PIB, capitale, continent, langue, etc.

2. Interface de consultation

Création d'une interface web intuitive permettant de parcourir les pays et d'accéder à leurs détails complets avec une présentation organisée de toutes les données.

3. Système de recherche et filtrage

Implémentation d'un système de recherche par nom et de filtrage par critères (continent, population, PIB) pour faciliter l'accès aux informations.

4. Validation et sécurité

Validation des données en entrée, protection contre les injections SQL et respect des bonnes pratiques de sécurité web.
